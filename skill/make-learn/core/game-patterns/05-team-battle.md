# 패턴 05 — 팀 대항 점수판 (퀴즈쇼)

학급을 팀으로 나눠 문제판에서 문제를 골라 풀고 점수를 겨루는 퀴즈쇼. **TV 화면·전자칠판 수업의 대표 패턴.**

## 이런 자료에 적합
- 단원 마무리 복습 등 **수업 전체를 활동으로 채울 때** (15~40분)
- 난이도를 섞을 수 있는 자료 (쉬움~어려움 문항이 골고루)
- 주 활용 방식이 'TV 화면 수업용' 또는 '전자칠판용'일 때 우선 추천. 개별 접속용에는 부적합(추천하지 않음)
- 문항 12~25개를 뽑을 수 있을 때

## 게임 규칙
1. 시작 화면에서 팀 수(2~6)와 팀 이름(기본: 동물 이모지 팀)을 설정.
2. **문제판**: 점수별 칸이 격자로 배치 (예: 100/200/300점 × 주제별 열). 높은 점수 = 어려운 문제.
3. 현재 차례 팀이 칸을 고르면 문제 표시 → 팀이 구두로 답함 → 교사가 '정답 공개' 클릭 → 교사가 [맞음/틀림] 판정 버튼 클릭.
4. 맞으면 점수 획득, 틀리면 0점(또는 감점 옵션) + 해설 표시. 푼 칸은 닫히고 다음 팀 차례.
5. 모든 칸 소진(또는 교사가 종료) 시 결과: 팀 순위 발표 (1위 축하 효과).
6. 재미 요소(선택): 일부 칸(전체의 15% 내외, 무작위 배치)에 보너스 숨김 — 칸을 열면 먼저 보너스가 공개된 뒤 문제가 정상 진행된다. 시작 화면에서 켬/끔.
   - ⭐ 점수 2배: 이 문제를 맞히면 칸 점수의 2배 획득
   - 💣 폭탄: 이 문제를 맞히면 칸 점수가 자기 팀 대신 **다음 차례 팀**에게 감 (틀리면 아무 일 없음)
   - 🎁 선물: 문제와 무관하게 즉시 +100, 이후 문제는 정상 진행

## 구현 포인트
- **조작자는 교사 1인**: 모든 버튼은 크고 명확하게, 진행에 필요한 버튼만 노출 (칸 선택 → 정답 공개 → 맞음/틀림 → 문제판 복귀).
- 점수판(팀 이름·점수·현재 차례)은 항상 화면 상단에 고정. 점수 변화는 카운트업 애니메이션.
- 교사 판정 방식이므로 정답 비교 로직이 필요 없음 → 단답·서술형 문항도 자유롭게 출제 가능 (이 패턴의 강점).
- 실수 대비: '점수 수정' 기능 (팀 점수 탭 → ±100 조정), '이전 화면' 복귀.
- 새로고침 사고 대비: 진행 상태(점수·닫힌 칸)를 localStorage에 저장, 재접속 시 "이어서 하시겠어요?" 제안.
- 팀 수 변경에 따라 점수판 레이아웃이 유연하게 (2~6팀).

## 활용 방식별 변형
- **TV 수업용**: 위 기본 규칙 그대로 (이 패턴의 기본형).
- **전자칠판용**: 학생이 직접 나와 칸을 고르고 정답 공개까지 터치. 버튼을 화면 하단에 크게.
- **개별 접속용**: 부적합. 교사가 원하면 동일 문항으로 패턴 01(골든벨 퀴즈) 개인판을 따로 만들 것을 제안.

## 검증 추가 포인트
- 모든 칸이 닫혔을 때 자동 종료 + 동점 처리 표시 (B5 연계)
- 차례 순환이 팀 수와 무관하게 정확한가
- 점수 수정·이어하기 동작 (localStorage try/catch — C5 연계)
- 높은 점수 문항이 실제로 더 어려운가 (난이도 배치 — A4 연계)
