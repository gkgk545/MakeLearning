# 패턴 02 — 카드 짝맞추기

'용어'와 '뜻'(또는 그림·기호·번역) 카드를 짝지어 찾는 게임. 뒤집기(기억력)형과 펼침(매칭)형 두 변형이 있다.

## 이런 자료에 적합
- **짝 관계가 명확한 자료**: 용어-정의, 단어-뜻(영어), 인물-업적, 기호-이름(음악 셈여림, 지도 기호), 나라-수도, 작품-작가
- 짝을 6~12쌍 뽑을 수 있을 때 (한 판 기준; 더 많으면 라운드로 분할)

## 변형 선택
- **뒤집기형(메모리)**: 모든 카드가 뒤집혀 있고 2장씩 열어 짝 찾기. 기억 요소가 있어 놀이성↑. 저학년·복습 초기에 적합.
- **펼침형(매칭)**: 왼쪽 열(용어)·오른쪽 열(뜻)이 모두 보이는 상태에서 하나씩 탭해 연결. 인지 부담이 학습 내용에 집중됨. 고학년·내용이 어려울 때 적합.

## 게임 규칙 (뒤집기형 기준)
1. 카드가 격자로 배치 (쌍 수×2장, 예: 8쌍 → 4×4).
2. 2장을 열어 짝이면 카드가 열린 채 고정(+점수), 아니면 **두 카드 각각의 올바른 짝(a↔b)을 한 줄로 잠깐 표시**한 뒤 다시 뒤집힘 (절대 규칙 '오답 시 정답 표시'의 매칭형 해석 — 이것이 학습 노출의 핵심).
3. 짝을 맞출 때마다 **해당 쌍의 의미를 보여주는 피드백** (예: "광합성 = 빛으로 양분을 만드는 작용 ✓").
4. 모든 짝 완성 시 결과: 걸린 시간·시도 횟수, 전체 짝 목록 복습 표시.

## 구현 포인트
- 카드 배치는 매 판 셔플. a카드와 b카드는 시각적으로 구분 (색·테두리).
- 뒤집기형: 2장 열린 상태에서 추가 클릭 잠금. 불일치 시 1초 정도 보여준 후 뒤집기 (학습 노출 시간).
- 펼침형: 첫 선택 카드 하이라이트 → 두 번째 선택 시 판정. 오답이면 흔들림 효과 + 정답 짝 잠깐 표시.
- 긴 텍스트 대비: 카드 안 글자 자동 축소(또는 말줄임 + 짝 완성 시 전문 표시).
- 라운드 분할 기준: 펼침형은 12쌍까지 한 판 가능. **뒤집기형은 카드 크기 확보를 위해 데스크톱 8쌍·모바일 5~6쌍을 넘으면 라운드제로 분할** (라운드당 5~8쌍).

## 활용 방식별 변형
- **TV 수업용**: 교사가 클릭, 학생들이 "○번이요!" 외치는 방식. 카드에 번호 표시 필수. 팀 대항 옵션: 맞춘 짝마다 현재 팀 점수 (턴 표시).
- **개별 접속용**: 기본 규칙. 모바일에서 카드 최소 크기 확보 (쌍 수가 많으면 펼침형 권장).
- **전자칠판용**: 카드 크게, 탭 방식이라 그대로 적합. 두 학생이 번갈아 나와 한 쌍씩 도전하는 운영 팁 안내.

## 검증 추가 포인트
- 잘못된 짝이 '맞음' 처리될 가능성 0 (a-b 매핑 id로 판정하는지) — A5 연계
- 뜻 카드 텍스트가 카드 크기를 넘치지 않는가 (C3 연계)
