# 패턴 06 — 낱말·빈칸 채우기

문장의 빈칸을 채우거나 흩어진 글자로 낱말을 완성하는 언어 중심 게임.

## 이런 자료에 적합
- **핵심 용어·표현 암기가 목표인 자료** — 핵심 개념어(전 교과), 영어 단어·문장, 국어 어휘·속담·맞춤법, 한자어, 사회·과학 용어
- 문장 속 맥락과 함께 익혀야 하는 내용 (빈칸형의 강점)
- 빈칸 문장 또는 목표 낱말을 8개 이상 뽑을 수 있을 때

## 변형 선택
- **빈칸 채우기형**: 자료의 핵심 문장에서 용어를 빈칸으로 — "식물은 ___에서 양분을 만든다". 답 입력은 ①글자 타일 조합(기본, 저·중학년) ②직접 타이핑(고학년·영어 철자 연습).
- **낱말 완성형**: 뜻·힌트를 보고 흩어진 글자 타일로 낱말 조합. 초성 힌트(한글)·첫 글자 힌트(영어) 단계 제공.

## 게임 규칙 (글자 타일 기준)
1. 빈칸 문장(또는 뜻 힌트)과 글자 타일들이 제시된다. 타일에는 정답 글자 + 함정 글자 2~3개 포함.
2. 타일을 순서대로 탭해 답을 조합. 타일 재탭으로 취소 가능.
3. '확인' 시 채점 — 정답: 완성 문장 전체를 강조 표시(맥락 복습). 오답: 2회까지 재도전(힌트 1개 공개), 이후 정답·해설 표시.
4. 전체 완료 시 결과: 점수, 힌트 사용 수, 전체 낱말·문장 목록 복습.

## 구현 포인트
- 한글 타일은 **음절 단위**(글자 단위)로 분해한다. 자모 조합 입력기는 만들지 않는다 (복잡도 대비 효과 낮음). 영어는 알파벳 단위.
- 직접 타이핑 모드 채점: 앞뒤 공백 제거, 한글은 그대로 비교, 영어는 대소문자 무시. 유사 오답("잎"vs"잎사귀")은 복수 정답 배열 허용 — `answer: ["잎", "잎사귀"]`.
- 모바일 타이핑 모드: 입력창 포커스 시 키보드가 화면을 가리므로 문제를 상단에 고정.
- 함정 타일은 자료에 나온 형태와 비슷한 글자로 (학습적 변별 연습이 되도록).

## 활용 방식별 변형
- **TV 수업용**: 빈칸 문장을 띄우고 학생들이 외치면 교사가 '글자 공개'를 단계적으로 클릭 (힌트 쇼 방식). 또는 모둠 칠판에 쓰게 한 뒤 정답 공개.
- **개별 접속용**: 기본 규칙 그대로. 타일 모드가 기본.
- **전자칠판용**: 타일 모드 + 대형 타일. 학생이 나와서 직접 조합.

## 검증 추가 포인트
- 타일 조합 취소·재조합 동작 (B4 연계)
- 복수 정답 처리, 타이핑 공백·대소문자 처리 (A2 연계)
- 함정 글자가 우연히 또 다른 올바른 답을 만들 수 없는가 (A5 연계)
