# 공유·활용 안내 가이드 (7단계)

게임 완성 후 교사에게 전달할 안내 모음. **주 활용 방식에 해당하는 절만 골라** 자연스럽게 풀어 안내한다 (문서 전체를 붙여넣지 않는다).

## 공통 — 파일 여는 방법

- 받은 `.html` 파일을 **더블클릭**하면 브라우저(크롬, 엣지 등)에서 바로 열린다. 설치·인터넷 필요 없음.
- 잘 안 열리면: 파일에 마우스 오른쪽 클릭 → 연결 프로그램 → Chrome 선택.

## ① TV 화면 수업용일 때

- 수업 전 교사 PC에 파일을 저장해두고 더블클릭으로 실행. 전체 화면은 `F11`.
- 수업 흐름 팁: 문제를 띄우고 학생들이 말로 답하게 한 뒤 '정답 공개' 버튼 클릭 → 해설 함께 읽기.

## ② 학생 개별 접속용일 때

가장 쉬운 방법부터 순서대로 안내한다.

**방법 1 — 파일 그대로 전달 (가장 쉬움, 추천)**
이 게임은 인터넷 없이도 동작하므로 파일 자체를 보내면 된다.
- 구글 클래스룸·e학습터·하이러닝·학급 밴드·카카오톡 등에 HTML 파일을 첨부 → 학생이 다운로드 후 열기.
- 스마트기기에서 다운로드한 파일이 바로 안 열리면: 파일 앱에서 해당 파일을 길게 눌러 'Chrome으로 열기'.

**방법 2 — 진짜 링크(URL)로 공유: Netlify Drop**
링크 클릭만으로 접속하게 하려면 무료 호스팅에 올린다.
1. [https://app.netlify.com/drop](https://app.netlify.com/drop) 접속 (무료 계정 필요, 구글 계정으로 가입 가능)
2. HTML 파일이 든 폴더를 화면에 끌어다 놓기 (파일명을 `index.html`로 바꿔서 폴더에 넣으면 주소가 깔끔해짐)
3. 생성된 링크(예: `https://○○○.netlify.app`)를 학생에게 공유 — QR코드로 만들면 더 편함

**방법 3 — 장기 운영: GitHub Pages**
학기 내내 여러 게임을 모아 운영하려면 GitHub Pages가 적합. 설정이 다소 복잡하므로 "도와드릴까요?"라고 묻고 원하면 단계별로 안내한다.

⚠️ 주의: 링크로 공유된 페이지는 누구나 볼 수 있다. 게임 문항에 학생 실명·개인정보를 넣지 않는다.

결과 확인 팁(게임에 이름 입력이 있는 경우): 기록은 각 학생 기기에만 남으므로, 결과 화면을 교사에게 보여주거나 캡처해 클래스룸·밴드에 제출하게 하면 확인이 편하다고 안내한다.

## ③ 전자칠판용일 때

- USB 또는 클라우드(구글 드라이브 등)로 파일을 전자칠판 PC에 옮긴 뒤 더블클릭 실행. 전체 화면 `F11`.
- 전자칠판이 인터넷이 되면 방법 2(Netlify Drop) 링크로 접속해도 된다.
- 터치 반응이 둔하면 전자칠판의 '터치 모드'(PC 터치) 설정을 확인하라고 안내.

## 문항 수정 방법 안내 (항상 포함)

다음 요지로 안내한다:

> 문항을 직접 바꾸실 수 있어요. 파일을 **메모장**(Windows: 오른쪽 클릭 → 연결 프로그램 → 메모장 / Mac: 텍스트 편집기)으로 열면 위쪽에 `✏️ 문항 수정 구역`이 있습니다. 따옴표 안의 문제·정답·해설 등의 글자만 바꾸고 저장한 뒤 다시 열면 적용됩니다. 따옴표(")와 쉼표(,)는 지우지 마세요. 수정이 번거로우면 저에게 "○번 문제를 ○○로 바꿔줘"라고 하셔도 됩니다.
