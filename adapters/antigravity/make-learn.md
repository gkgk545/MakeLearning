# make-learn — 학습자료로 수업용 웹앱 게임 만들기

1. 홈 디렉토리의 `~/.make-learn/SKILL.md` 파일을 읽는다 (없으면 사용자에게 "make-learn 설치 스크립트를 먼저 실행해 주세요"라고 안내하고 중단).
2. SKILL.md의 지침과 그것이 가리키는 `~/.make-learn/core/` 문서들을 따라 7단계 파이프라인(입력 수집 → 자료 분석 → 순차 인터뷰 → 설계 계획 → 제작 → 검증 → 반환)을 끝까지 수행한다.
3. 모든 대화는 한국어로, 교사(비개발자) 눈높이로 진행한다.
4. 사용자가 이 워크플로우와 함께 학습자료를 첨부했거나 경로를 알려줬다면 그것을 1단계 입력으로 사용한다.
