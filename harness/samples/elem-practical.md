# [샘플 학습자료] 초등 5학년 실과 — 균형 잡힌 식사와 식품군

> 단원 학습 정리

## 1. 식품의 다섯 가지 기초 식품군 (우리나라 기준 6가지 중 수업 범위 5가지)

| 식품군 | 주요 영양소 | 우리 몸에서 하는 일 | 식품 예 |
|---|---|---|---|
| 곡류 | 탄수화물 | 활동에 필요한 힘(에너지)을 줘요 | 밥, 빵, 국수, 감자, 고구마 |
| 고기·생선·달걀·콩류 | 단백질 | 근육과 몸을 만들어요 | 소고기, 고등어, 달걀, 두부, 콩 |
| 채소류 | 비타민, 무기질 | 몸의 기능을 조절해요 | 시금치, 당근, 오이, 배추 |
| 과일류 | 비타민, 수분 | 몸의 기능을 조절하고 피로를 풀어줘요 | 사과, 귤, 포도, 바나나 |
| 우유·유제품류 | 칼슘 | 뼈와 이를 튼튼하게 해요 | 우유, 치즈, 요구르트 |

## 2. 균형 잡힌 식사란?

- 다섯 가지 식품군을 **골고루**, **알맞은 양**으로 먹는 것.
- 한 가지 식품군만 많이 먹으면 영양이 한쪽으로 치우쳐요.
- 식단을 짤 때는 식품군이 골고루 들어갔는지 확인!

## 3. 식단 평가 연습

- 식단 A: 밥 + 고등어구이 + 시금치나물 + 사과 + 우유 → 다섯 식품군이 모두 포함된 균형 잡힌 식단 ⭕
- 식단 B: 라면 + 빵 + 감자튀김 → 곡류에 치우침 ❌ (단백질·채소·과일·유제품 부족)

## 확인 문제

1. 밥과 국수는 어느 식품군? → 곡류
2. 근육과 몸을 만드는 영양소는? → 단백질
3. 두부는 어느 식품군? → 고기·생선·달걀·콩류
4. 뼈와 이를 튼튼하게 하는 식품군은? → 우유·유제품류
5. 당근과 배추는 어느 식품군? → 채소류
6. 균형 잡힌 식사란? → 다섯 식품군을 골고루, 알맞은 양으로 먹는 것
