# [샘플 학습자료] 초등 4학년 수학 — 분수의 덧셈과 뺄셈 (분모가 같은 경우)

> 수업 정리 학습지 발췌

## 오늘 배운 내용

1. **분모가 같은 분수의 덧셈**: 분모는 그대로 두고 분자끼리 더한다.
   - 예: 2/7 + 3/7 = 5/7
   - 예: 3/8 + 4/8 = 7/8
2. **분모가 같은 분수의 뺄셈**: 분모는 그대로 두고 분자끼리 뺀다.
   - 예: 5/6 − 2/6 = 3/6
   - 예: 7/9 − 3/9 = 4/9
3. **계산 결과가 가분수이면 대분수로 나타낼 수 있다.**
   - 예: 4/5 + 3/5 = 7/5 = 1과 2/5
4. **1에서 분수 빼기**: 1을 분모가 같은 분수로 바꾼 뒤 뺀다.
   - 예: 1 − 1/4 = 4/4 − 1/4 = 3/4

## 자주 하는 실수 (주의!)

- 분모끼리 더하면 안 돼요. 2/7 + 3/7 = 5/14 (❌) → 5/7 (⭕)
- 분자와 분모를 함께 빼면 안 돼요.
