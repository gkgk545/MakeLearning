# [샘플 학습자료] 중학교 1학년 과학 — 광합성

> 단원 정리 노트

## 1. 광합성이란?

식물이 **빛에너지**를 이용하여 물과 이산화 탄소로 **양분(포도당)**을 만드는 과정.

- 일어나는 장소: 잎의 **엽록체** (초록색 색소인 **엽록소**가 빛을 흡수)
- 광합성식: 물 + 이산화 탄소 --(빛에너지)--> 포도당 + 산소

## 2. 광합성에 필요한 것 / 만들어지는 것

| 필요한 것 | 만들어지는 것 |
|---|---|
| 물 (뿌리에서 흡수) | 포도당 (양분) |
| 이산화 탄소 (잎의 기공으로 흡수) | 산소 (기공으로 방출) |
| 빛에너지 | |

## 3. 광합성에 영향을 주는 환경 요인

1. **빛의 세기**: 셀수록 광합성량 증가 (일정 수준 이상에서는 더 증가하지 않음)
2. **이산화 탄소 농도**: 높을수록 증가 (일정 수준 이상에서는 일정)
3. **온도**: 적정 온도까지 증가, 너무 높으면 급격히 감소

## 4. 관련 실험·확인 방법

- 포도당은 곧 **녹말**로 바뀌어 잎에 저장 → **아이오딘-아이오딘화 칼륨 용액**을 떨어뜨리면 청람색으로 변함 (녹말 검출)
- 검정말 실험: 빛을 받은 검정말에서 발생하는 기포 = **산소**
- BTB 용액: 이산화 탄소가 많으면 노란색, 적으면 파란색 (광합성으로 CO₂가 소모되면 노란색 → 파란색)

## 5. 식물의 호흡과 비교

| | 광합성 | 호흡 |
|---|---|---|
| 일어나는 때 | 빛이 있을 때 | 항상 (낮+밤) |
| 기체 출입 | CO₂ 흡수, O₂ 방출 | O₂ 흡수, CO₂ 방출 |
| 에너지 | 양분에 에너지 저장 | 양분을 분해해 에너지 얻음 |
