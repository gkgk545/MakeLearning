# [샘플 학습자료] 초등 3학년 국어 — 헷갈리기 쉬운 맞춤법

> 받아쓰기 대비 정리 자료

## 오늘 배운 맞춤법

| 맞는 표기 ⭕ | 틀린 표기 ❌ | 도움말 |
|---|---|---|
| 어떻게 | 어떡해(다른 뜻) | "어떻게 가요?" / "나 어떡해"는 '어떻게 해'의 준말 |
| 됐어요 | 됬어요 | '되었어요'의 준말이라 '됐어요' |
| 안 돼요 | 안 되요 | '돼요'는 '되어요'의 준말 |
| 며칠 | 몇일 | '몇 일'로 쓰지 않아요 |
| 깨끗이 | 깨끗히 | 'ㅅ' 받침 뒤에는 '-이' |
| 곰곰이 | 곰곰히 | 첩어 뒤에는 '-이' |
| 가르치다 | 가리치다 | 선생님이 공부를 가르치다 |
| 가리키다 | 가르키다 | 손가락으로 방향을 가리키다 |
| 잃어버리다 | 잊어버리다(다른 뜻) | 물건은 잃어버리고, 기억은 잊어버려요 |
| 낫다 | 낮다(다른 뜻) | 병이 낫다 / 키가 낮다... 아니죠, 건물이 낮다! |

## 문장으로 연습하기

1. 숙제를 (어떻게/어떡해) 하면 좋을까? → 어떻게
2. 이제 청소 다 (됐어요/됬어요). → 됐어요
3. 복도에서 뛰면 (안 돼요/안 되요). → 안 돼요
4. 방학이 (며칠/몇일) 남았지? → 며칠
5. 교실을 (깨끗이/깨끗히) 청소하자. → 깨끗이
6. 선생님이 수학을 (가르쳐/가리켜) 주셨다. → 가르쳐
7. 지도에서 우리 학교를 (가리켜/가르쳐) 보세요. → 가리켜
8. 지우개를 (잃어버렸다/잊어버렸다). → 잃어버렸다
9. 감기가 다 (나았다/낳았다). → 나았다
10. (곰곰이/곰곰히) 생각해 보았다. → 곰곰이
